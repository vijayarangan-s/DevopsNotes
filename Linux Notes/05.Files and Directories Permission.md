### Symbolic (rwx)

| Symbol | Meaning          |
|--------|------------------|
| `r`    | Read             |
| `w`    | Write            |

| `-`    | No permission    |

### Categories

| Symbol | Category         |
|--------|------------------|

| `g`    | Group            |
| `o`    | Others           |
| `a`    | All (u+g+o)      |

### ğŸ”¹ Symbolic Mode


### ğŸ”¹ Numeric (Octal) Mode

| Number | Permission |
|--------|------------|

### Format:
```bash

```

### Examples:

```bash

```

## ğŸ“ Directories vs Files

- **Execute (`x`) on directories** = permission to enter (`cd`) the directory
- **Read (`r`)** = see the contents with `ls`
- **Write (`w`)** = create/delete inside the directory
